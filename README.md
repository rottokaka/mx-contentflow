积录 - 趁你还未忘记
======================================================================================
> 趁你还未忘记，赶紧积录一下。所谓积录，积少成多，目之以录。
  把碎片化的知识，智能化，体系化，组织化，形成内容产品，方便完整自己的知识图谱，维护公共的知识库。
  包括知识准确性、重复性、传递性，尊重个人成果、知识共享，以及知识付费等概念。
